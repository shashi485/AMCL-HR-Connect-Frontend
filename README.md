# AMCL HR Connect – Frontend

Frontend for the AMCL HR Connect Employee Management System.

Live Site:
https://amcl-hr-connect-frontend1.vercel.app

Backend API:
https://amcl-hr-connect-backend.onrender.com

Tech Stack:
- React
- Tailwind CSS
- Axios

Features:
- Login & Registration
- Dashboard
- Attendance
- Leave Management
- Performance Reviews
- Messaging

Run locally:
npm install
npm start
